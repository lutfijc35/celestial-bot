# Template Pengumuman Fitur Baru

Copy template di bawah, isi placeholder `{...}`, paste ke Discord.

---

## Template

```
{EMOJI} FITUR BARU: {NAMA FITUR} {EMOJI}

Halo @everyone! Ada fitur baru di server — **{deskripsi singkat fitur}**

━━━━━━━━━━━━━━━━━━━━

**Gimana cara kerjanya?**

1️⃣ {Langkah 1}
2️⃣ {Langkah 2}
3️⃣ {Langkah 3}
4️⃣ {Langkah 4 — hasil/hadiah}

━━━━━━━━━━━━━━━━━━━━

**📌 Yang perlu diketahui:**
• {Catatan 1}
• {Catatan 2}
• {Catatan 3}
• Ketik `{/command}` untuk {deskripsi command}

━━━━━━━━━━━━━━━━━━━━

{Kalimat penutup ajakan}
```

## Contoh: Starboard

```
⭐ FITUR BARU: TAMBAK GARAM ⭐

Halo @everyone! Ada fitur baru di server — **Tambak Garam & Hall of Fame!**

━━━━━━━━━━━━━━━━━━━━

**Gimana cara kerjanya?**

1️⃣ Kirim pesan kamu di #tambak-garam — bisa text, gambar, meme, apapun
2️⃣ Kalau pesan kamu **dapet 5 react ⭐** dari member lain...
3️⃣ Pesan kamu otomatis masuk ke #orang-hitam sebagai **Hall of Fame** 🏆
4️⃣ Kamu dapet **role spesial selama 30 hari** sebagai hadiah!

━━━━━━━━━━━━━━━━━━━━

**📌 Yang perlu diketahui:**
• React ⭐ (bintang) pada pesan yang menurut kamu layak masuk hall of fame
• Butuh minimal **5 bintang** dari member berbeda
• Role spesial otomatis dicabut setelah **30 hari**
• Ketik `/leaderboard` untuk lihat **top 10 ranking** member paling sering masuk hall of fame

━━━━━━━━━━━━━━━━━━━━

Ayo mulai garam-garamin pesan teman kalian! ⭐
```

## Contoh: Registrasi Akun

```
📝 FITUR BARU: REGISTRASI AKUN E7 📝

Halo @everyone! Sekarang kamu bisa **daftarkan akun Epic Seven** di server ini!

━━━━━━━━━━━━━━━━━━━━

**Gimana cara kerjanya?**

1️⃣ React ✅ di #rules untuk unlock channel
2️⃣ Buka #register-here dan klik tombol **Daftar Sekarang**
3️⃣ Isi form: Server, Guild (opsional), Nickname
4️⃣ Tunggu admin approve — kamu akan dapat role guild otomatis!

━━━━━━━━━━━━━━━━━━━━

**📌 Yang perlu diketahui:**
• Nickname harus unik per server (tidak boleh sama dengan member lain)
• Kamu bisa daftar lebih dari 1 akun (misal Asia + Global)
• Ketik `/profile` untuk lihat akun yang sudah terdaftar
• Ketik `/edit` untuk ubah data akun

━━━━━━━━━━━━━━━━━━━━

Yuk langsung daftar akun E7 kamu! 🎮
```

---

## Tips

- Ganti `@everyone` dengan `@here` jika tidak mau ping semua member
- Gunakan `#channel-name` langsung di Discord untuk auto-link ke channel
- Emoji di awal dan akhir judul supaya eye-catching
- Maksimal 4 langkah supaya tidak terlalu panjang
- Kalimat penutup harus ajakan (call to action)
