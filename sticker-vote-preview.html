Part 1 — Submit Flow
Member upload sticker baru → community vote → admin approve → upload ke server
📋 Aturan Main
  • Siapa: Semua member (cooldown 1 submit / 24 jam)
  • File: PNG / APNG / Lottie · max 512KB · ideal 320×320
  • Durasi voting: 7 hari
  • Lolos: Net score (👍 − 👎) ≥ 5 dan total voter ≥ 10
  • Mode: Semi-auto — admin approve final sebelum upload ke server
  • Vote: Bisa ganti pilihan, list voter anonim (hanya total)
1️⃣ Member submit sticker — /submit-sticker
Slash command dengan attachment:
/submit-sticker  sticker:[📎 cute-cat.png]  nama:Cute Cat  tag:😺

Part 2 — Retention Flow
Admin pick sticker existing → community vote keep/remove → admin confirm → delete
📋 Aturan Retention Polling
  • Trigger: Admin manual via /poll-sticker-keep (autocomplete dari sticker existing)
  • Durasi voting: 7 hari
  • Lolos "Remove": Net (🗑️ − 💚) ≥ 5 dan total voter ≥ 10
  • Mode: Semi-auto — admin confirm final sebelum delete
  • 1 sticker: max 1 retention poll aktif (tidak bisa trigger ulang kalau ada yang aktif)
1️⃣ Admin trigger retention poll — /poll-sticker-keep
Slash command dengan autocomplete sticker existing:
