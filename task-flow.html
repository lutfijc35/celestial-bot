📋 Alur Lengkap
👑 Admin ketik /promote Isi judul, deskripsi, harga → embed + button muncul
↓
👤 User klik tombol "📩 Saya Tertarik" Modal form muncul: IGN, detail request, budget
↓
🤖 Bot kirim embed ke #task-joki Info requester + detail + button "👤 Assign" + mention @role
↓
📨 Bot DM ke user "Request kamu sudah dikirim, admin akan menghubungi"
↓
👑 Admin klik "👤 Assign" di #task-joki Pilih member yang akan jadi joki dari dropdown
↓
🔒 Bot buat private channel #task-{id} Visible: requester + joki + admin saja
↓
📋 Bot kirim info task di channel baru Detail request + mention requester & joki
↓
📨 Bot DM ke requester "Task kamu di-assign ke @joki, cek #task-{id}"
Requester & joki chat di private channel↓
👑 Admin ketik /close-task di channel Saat task selesai
↓
📊 Bot kirim summary ke #task-joki Requester, joki, durasi task
↓
🗑️ Channel #task-{id} dihapus Task ditandai closed di database
